Python Fundamentals

Module 1: Algorithms and Flowcharting

Programming Concepts Defined

  • Computer Program: List of instructions that directs the computer on its operations (e.g., Calculator, Games, Websites)
  • Programming: The act of developing a computer program
  • Programmer: The person who develops computer programs

Steps in Program Planning and Development

  1. Problem Analysis: Define and understand the problem, analyze requirements, constraints, and objectives
  2. Setting Up the Algorithm: Break down the problem into smaller tasks, decide logical flow
  3. Coding: Translate algorithm into Python code using proper syntax
  4. Encoding: Save the code into a .py file
  5. Running, Testing, and Debugging: Test with different inputs, fix issues
  6. Documentation: Create explanatory materials and comments

Example: Sum of 3 Numbers

Problem Analysis:
Input: 3 numbers (num1, num2, num3)
Output: Sum of the 3 numbers (sum)
Process: Sum = num1 + num2 + num3

Algorithm:
1. Begin
2. Read num1, num2
3. Sum = num1 + num2
4. Read num3
5. Sum = sum + num3
6. Print sum
7. End

Flowchart Symbols

  • Terminal Symbol: Start/end of flowchart
  • Input/Output Symbol: For input and output operations
  • Processing Symbol: For processing and computations
  • Decision Symbol: For YES/NO or TRUE/FALSE paths

Module 2: Introduction to Python Programming

Why Use Python?

  • Easy to learn and use with simple, intuitive syntax
  • Versatile (web development, data analysis, AI, scientific computing, etc.)
  • Large community and ecosystem with many libraries
  • Cross-platform support (Windows, macOS, Linux)
  • High-level language (more human-readable than low-level languages)
  • Interpreted language (no need to compile before executing)

Variables and Data Types

Variables: Used to store values in memory

name = "python"  # 'name' is variable, 'python' is value

Naming Variables Rules

  • Can contain letters, digits, and underscore (_)
  • Must begin with a letter
  • Case-sensitive (myVar ≠ myvar)
  • Cannot be Python keywords (if, else, while, etc.)

Data Types

  • String: Text in single/double quotes ('hello', "world")
  • Integer: Whole numbers (5, -3, 100)
  • Float: Numbers with decimals (3.14, -0.5)
  • Boolean: True or False values

Print Function

print("Hello, World!")  # Basic print
print("Hello", "World", sep="-")  # With separator
print("Hello", end=" ")  # Change end character
print("World")

Input Function

name = input("Enter your name: ")
print("Hello", name)

Casting

x = "10"
y = int(x)  # Convert to integer
z = float(x)  # Convert to float
s = str(10)  # Convert to string

Module 3: Decision Making Statements

Comparison Operators

  • == (equal to)
  • != (not equal to)
  • > (greater than)
  • < (less than)
  • >= (greater than or equal to)
  • <= (less than or equal to)

Logical Operators

  • and (True if both conditions are true)
  • or (True if at least one condition is true)
  • not (Reverses the result)

If Statement

number1 = 5
number2 = 7
if number1 < number2:
    print("The first number is smaller")

If-Else Statement

num = 3
if num >= 0:
    print("Positive or Zero")
else:
    print("Negative number")

If-Elif-Else Statement

num = 3.4
if num > 0:
    print("Positive number")
elif num == 0:
    print("Zero")
else:
    print("Negative number")

Nested If-Else

if num >= 0:
    if num == 0:
        print("Zero")
    else:
        print("Positive number")
else:
    print("Negative number")

Module 4: Iterative Statements (Loops)

While Loop

i = 1
while i < 6:
    print(i)
    i += 1

For Loop

# Iterating through a string
for x in "banana":
    print(x)

# Using range
for x in range(6):  # 0 to 5
    print(x)

for x in range(2, 6):  # 2 to 5
    print(x)

for x in range(2, 6, 2):  # 2, 4 (step by 2)
    print(x)

Loop Control Statements

# Break statement
for x in "python":
    if x == "h":
        break
    print(x)  # Output: p, y, t

# Continue statement
for x in "python":
    if x == "h":
        continue
    print(x)  # Output: p, y, t, o, n

Else in Loops

for x in range(2, 6):
    print(x)
else:
    print("Finally finished!")

Nested Loops

for i in range(2, 6):
    print(f"First 10 multiples of: {i}")
    for j in range(1, 11):
        print(i * j, end=", ")
    print("\n")

Module 5: Functions

Defining a Function

def greet():
    print("Hello World")

greet()  # Call the function

Function with Parameters

def add(num1, num2):
    sum = num1 + num2
    return sum

result = add(1, 2)
print(result)  # Output: 3

Default Parameters

def greet(name="Joy"):
    print(f"Hi {name}")

greet()  # Output: Hi Joy
greet("Eyren")  # Output: Hi Eyren

Built-in Functions

  • max() - Gets the maximum value
  • min() - Gets the minimum value
  • sum() - Gets the sum
  • len() - Gets the length

Module 6: Python Lists (Arrays)

Creating Lists

food = ['cake', 'burger', 'fries']
print(food[0])  # Output: cake
print(food[-1])  # Output: fries (last item)

Modifying Lists

food[0] = 'pizza'  # Change first item
print(food)  # Output: ['pizza', 'burger', 'fries']

List Methods

# append() - Add item to end
food.append('ice cream')

# insert() - Add item at specific position
food.insert(1, 'salad')

# remove() - Remove first occurrence
food.remove('burger')

# pop() - Remove item at index
food.pop(0)

# sort() - Sort the list
food.sort()

# reverse() - Reverse the list
food.reverse()

# extend() - Add elements from another list
drinks = ['coke', 'juice']
food.extend(drinks)
